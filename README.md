# Huffman Code with Fibonacci Numbers — TOR III Project

## Overview
This project demonstrates **lossless data compression** using **Huffman coding**.  
It explores how symbol frequency affects code length and overall compression efficiency.

## Objectives
- Construct a Huffman code using frequencies derived from the first eight Fibonacci numbers:  
  `a:1, b:1, c:2, d:3, e:5, f:8, g:13, h:21`
- Generate a dataset of 108 symbols following these frequencies.
- Encode the dataset using Huffman codes and calculate the **compression ratio** compared to the uncoded case.
- Extend the alphabet to 2-symbol blocks and compare compression performance with single-symbol encoding.

## Deliverables
- Implementation of Huffman encoding and decoding.
- Demonstration of algorithm steps and results.
- Presentation showing:
  - Algorithm explanation  
  - Example encoding process  
  - Compression ratio comparison  
  - Observations and conclusions
